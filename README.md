# 翻译小助手

一个干净简洁、方便的chrome插件，你的翻译小助手，让你轻松阅读文章。

本插件是为了阅读外文时，不用再来回切换去翻译，也不用再担心全文一键翻译然后狗屁不通。在阅读时遇到不懂单词，然后让你快速翻译。

## 安装
chrome浏览器 ——> 更多工具(L) ——> 扩展程序(E) ——> 加载已解压的扩展程序
